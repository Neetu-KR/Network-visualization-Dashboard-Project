# 🌐 Network Visualization Dashboard

An interactive and aesthetic dashboard for visualizing network connections using Python, Streamlit, NetworkX, and Plotly.

---

# 📌 Project Overview

The Network Visualization Dashboard is a mini project designed to visualize relationships between connected nodes and edges in an interactive graphical format.

This project helps users understand:
- Network structures
- Node connections
- Graph layouts
- Relationship analysis

---

# 🚀 Features

✅ Interactive network graph

✅ Multiple graph layouts
- Spring Layout
- Circular Layout
- Kamada-Kawai Layout

✅ Real-time visualization

✅ Network statistics

✅ Aesthetic dark UI

✅ Sidebar controls

---

# 🛠 Technologies Used

| Technology | Purpose |
|---|---|
| Python | Programming Language |
| Streamlit | Dashboard UI |
| NetworkX | Network Graph Creation |
| Plotly | Interactive Visualization |
| Pandas | Data Handling |
